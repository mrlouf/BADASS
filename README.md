# BADASS
Bgp At Doors of Autonomous Systems is Simple (BADASS), aka NetPractice's ugly cousin on steroids.

## Description
BADASS is a project that aims at setting up three autonomous systems (AS) using GNS3 and Docker containers.

## Prerequisites
- GNS3: install following the instructions at https://docs.gns3.com/docs/getting-started/installation/
